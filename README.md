A self-bot Discord API wrapper for Go.

> [!NOTE]\
> discoself is currently in the alpha stages of development. Once the module reaches a stable release the relevant documentation will be created.

## Features

- **Fast** - discoself is written in **Go**, allowing it, and you, to leverage its *fearless concurrency*.
- **Easy to Integrate** - The API was designed with **simplicity** and **powerfulness** in mind. This allows you to focus on what matters; the self-bot.

## Installation

**discoself** can be installed by using *go get*:

```
go get github.com/krishnassh/discoself
```

After this command has been run, **discoself** is ready to use. **discoself** can be updated by using *go get -u*:

```
go get -u github.com/krishnassh/discoself
```
